# Benchmarks

Run live measurements in your browser: [crypto-hasher-benchmark](https://CRYPTO-HASHER.github.io/crypto-hasher-benchmark/).

Figures below were measured with **crypto-hasher 4.10.0** in **Chrome v131** on a **Ryzen 9 7900X**.

Two scenarios:

1. Throughput with the shorthand API — **32-byte** inputs
2. Throughput with the shorthand API — **1 MB** inputs

## MD5

| Library | 32 bytes | 1 MB |
| ------- | -------- | ---- |
| **crypto-hasher 4.10.0** | **110.52 MB/s** | **850.31 MB/s** |
| spark-md5 3.0.2 | 38.87 MB/s | 171.73 MB/s |
| md5-wasm 2.0.0 | 37.36 MB/s | 131.77 MB/s |
| crypto-js 4.1.1 | 9.30 MB/s | 46.71 MB/s |
| node-forge 1.3.1 | 18.23 MB/s | 28.94 MB/s |
| md5 2.3.0 | 14.50 MB/s | 21.65 MB/s |
