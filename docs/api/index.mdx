# API overview

Everything is imported from the package root:

```ts
import {
  md5,
  createSHA256,
  createHMAC,
  pbkdf2,
  argon2id,
  type IDataType,
  type IHasher,
} from "crypto-hasher";
```

## Common patterns

```ts
// One-shot → Promise<string> (hex)
await sha256(data);

// Streaming → IHasher
const h = await createSHA256();
h.init().update(data);
h.digest();           // string
h.digest("binary");   // Uint8Array

// KDF → string | Uint8Array depending on outputType
await argon2id({ ..., outputType: "encoded" });
```

All public hash and KDF entry points are **asynchronous** (they return `Promise`s) because WebAssembly module compilation/instantiation is async in many environments.
